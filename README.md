# Pong Game (Python, Turtle)

A classic **Pong game** implemented in Python using the `turtle` module.  
This version features:

- Left paddle controlled by the player (`W` and `S`)
- Right paddle controlled by a simple AI
- Scoreboard with a maximum score limit
- Ball that speeds up after paddle collisions
- Game over message when a player reaches the max score

![Gameplay Preview](assets/game_preview.gif)

---

## 🎮 Controls

- **Left paddle**:  
  - Move up → `W`  
  - Move down → `S`

- **Right paddle**: AI controlled automatically

---

## ⚙️ Requirements

- Python 3.10+ (recommended)
- No external libraries required (uses built-in `turtle`)
